Principal Components Analysis of Financial Data

Technologies: MATLAB

This project was an assignment for my Scientific Computing class. The task was to use SVD to perform PCA on two sets of data (each containing about thirty stocks' normalized prices over time). We then had to reconstruct the stock prices using the principal components, and calculating the RMS error of both sets of data, in order to determine which set of stocks were large cap, and which were small cap.
